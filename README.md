# Omnus Technologies Website

Professional website for [Omnus Technologies](https://omnustechnologies.com/) — an innovative infrastructure technology platform that empowers professional services providers to manage operations, compliance, and client engagement with greater efficiency and intelligence.

## Tech Stack

- Vanilla HTML, CSS, JavaScript (no build step required)
- Crimson Pro + DM Sans typography
- Custom design system with light/dark mode support
- CSS scroll-driven animations with IntersectionObserver fallback
- Responsive mobile-first layout

## Structure

```
├── index.html              # Main landing page
├── privacy-policy.html     # Privacy policy
├── legal-disclaimers.html  # Legal disclaimers
├── style.css               # Full design system & component styles
├── base.css                # CSS reset & foundation
├── app.js                  # Theme toggle, navigation, modals, animations
└── images/                 # Logo, backgrounds, team photos, OG image
```

## Development

No build tools needed — open `index.html` directly in a browser or serve with any static file server.
